The project directory is assumed to reside in:
	C:\Microchip Starter Kits\PIC18F Starter Kit 1\Demos

Set the include directories to search those listed in Appendix A of the pdf.

The linker file "rm18f46j50_g.lkr" is used for SD Bootloader compatibility. See the
Starter Kit User's Guide.

The hex file, "Level.hex" is a renamed version of PRGA_LevelDemo.hex.
